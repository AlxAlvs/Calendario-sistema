Instruções

- Abrir a pasta calendarioSis deste mesmo diretorio no eclipse(IDE).
- No arquivo application.properties configurar usuario e senha do banco de dados, caso haja.
- Executar a api como "Java application".
- Abrir a pasta calendarioSis dentro da pasta frontend deste mesmo diretorio pelo cmd. executar o commando "ng serve" para executar a aplicação.
- Abrir o pgadmin do banco de dados postgreSQL executar o seguinte script: 

CREATE DATABASE calendario;

CREATE TABLE calendario(
   id serial PRIMARY KEY,
   descricao VARCHAR (50) NOT NULL,
   hora_inicio VARCHAR (50) NOT NULL,
   hora_termino VARCHAR (50) NOT NULL
);

- Abrir a página inicial do sistema em http://localhost:4200
- Fazer login com o usuario: admin e a senha: admin
